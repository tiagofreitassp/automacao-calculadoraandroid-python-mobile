=== Automacao de calculos na Calculadora do Android ===

0.Objetivo
 .Realizar calculos basicos na calculadora do Android no emulador
 .Realizar calculos basicos na calculadora do Android no smartphone

1.Requisitos:
 .Python 3.7
 .PyCharm ou qualquer IDE/Bloco de Notas que suporte Python
 .Frameworks: Unittest, Appium, OS e Selenium (Instalar utilizando o PIP Python).

2.Configuracoes:
 .O projeto foi criado no MacOS e testado tambem no Windows. Noa e necessario alterar nada.
 .Os metodos para criar o driver Mobile estao na classe EmuladorDriver e MobileDriver

3.Execucao:
 .Abrir uma das classes ***Test.py no pyCharm
 .No caso para executar num smartphone, abra a classe Mobile Driver e altere o valor no campo "deviceName"

4.Evidencias:
 .Apos a execucao as imagens de evidencias sao armazenadas na pasta evidencias.

5.Massas:
 .Nao e necessario criar massas por enquanto.
